# Smart-Ranker Assignment (Candidate)

Do the TODOs in `frontend/src/ds/trie.js`, `backend/src/ds/lru.js`, and `backend/src/ds/ranker.js`.

Run locally:
1. Install workspace: `npm ci`
2. Start dev servers:
   - `cd backend && npm install && npm run dev`
   - `cd frontend && npm install && npm run dev`

Run tests:
- `cd backend && npm test`
- `cd frontend && npm test`

Deploy frontend to Vercel and backend to any host. Submit frontend URL (and backend URL if separate).
